# Geo-Physical Cosmic Aggregator

Dashboard cientifico con React, Tailwind, rutas protegidas y paneles de clima espacial.

Integraciones:
- NOAA SWPC (Kp, viento solar, campo magnetico, alertas)
- NASA POWER (irradiancia solar)
- WHERE THE ISS AT (posicion ISS)
- Met.no (fase y salida/puesta de la luna)
- Open-Meteo Marine (oleaje)

Secciones:
- Dashboard
- Datos Solares
- Rastreo ISS
- Luna
- Olas del Mar

## Scripts
- `npm run dev`
- `npm run build`
- `npm run preview`
