1. STACK: React (Vite), Tailwind CSS, React Router v6, Framer Motion y Recharts.
2. SIN PERSISTENCIA: Maneja todo el estado mediante Context API y sessionStorage. No uses bases de datos.
3. AUTENTICACIÓN:
   - Implementa un AuthContext que proteja las rutas '/dashboard' y '/observatory'.
   - La pantalla de login debe ser minimalista con una animación de fondo de partículas.
4. INTEGRACIÓN DE APIs (Lógica de Servicios):
   - NASA POWER API: Para obtener irradiancia solar.
   - NOAA SWPC: Para obtener el índice Kp actual y viento solar.
   - WHERE THE ISS AT: Para la latitud/longitud de la ISS.
5. COMPONENTES CLAVE:
   - Sidebar: Navegación entre Dashboard, Datos Solares y Rastreo ISS.
   - Dashboard: Un panel de control con 3 widgets principales.
   - Gráfica Interactiva: Usa Recharts para mostrar la fluctuación de la radiación solar.
   - Indicador de Alerta: Un componente que use Framer Motion para "latir" si hay actividad geomagnética alta.
6. MICRO-INTERACCIONES:
   - Las gráficas deben animarse al cargar (draw effect).
   - Usa AnimatePresence para transiciones suaves entre páginas.
   - Los botones deben tener un efecto de escala sutil (0.95) al hacer clic.

Genera primero el layout principal, el sistema de rutas protegidas y la integración de la API de NOAA para mostrar el estado del clima espacial en tiempo real.
